# 💰 Expense Tracker Backend API

A robust backend for Expense Tracker application built with **Node.js, Express, and MongoDB**.  
It supports authentication, expense management, and dashboard analytics.

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- dotenv

---

## 📁 Project Structure
Server/
│── config/
│ └── db.js
│
│── .env
│── server.js
│── package.json

---

## ⚙️ Installation

### 1. Clone repo
```bash
git clone https://github.com/shubham28052001/Expense-Tracker.git
 ```

## Install dependencies

npm install

---

## create .env file
PORT=5000
MONGO_URI=your_mongodb_connection_string

---

## Run server
npm start
